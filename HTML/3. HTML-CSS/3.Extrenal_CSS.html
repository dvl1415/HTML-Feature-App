*** Priority for the CSS Style:
Base on the order, how we put in the HTML File. I mean by that is
if we put Internal, External Declaration and then Inline in the HTML File
then The priority will be given to the Inline CSS
so, In a nutshell priority base on the two following things:
1. Based on the order how you declared the styles in the HTML File
2. or the Priority is Inline > Internal > External
